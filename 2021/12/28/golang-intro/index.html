Introduction

Go was designed at Google in 2007 by Robert Griesemer, Rob Pike and Ken Thompson. It was announced to the public in November 2009

Go is a statically typed, compiled language. It was aimed to improve programming productivity.

It has syntactic similarity to C, but includes memory safety, garbage collection, static typing, readability and built-in concurrency support.

It has simple, consistent features, which prefer maintainability over expressiveness.

Why Go?

It is Open Source, Free and Cross-platform.

Well established / stable / Widely Adopted / Extensive Support

Efficient Runtime - many good features of other languages

Concurrency mechanisms - Goroutines and Channels

Extensive IDE Support

Well supported by Google Go Toolchain - use of the go command tool
